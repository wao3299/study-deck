# 仮免学科試験デック追加 設計

日付: 2026-07-17

## 目的

日本の普通自動車免許・仮免学科試験（第一段階）の対策デックを追加する。将来的に第二段階（本免範囲）の問題を同じデックに追加できる構成にする。

## デック定義

- コード: `MENKYO`、URL: `/study-deck/menkyo/`、データ: `public/questions-menkyo.json`
- 新規ファイル `src/pages/menkyo.astro`（owasp.astro と同型）:
  - eyebrow: `運転免許 · 仮免学科（第一段階）`
  - title: `仮免学科試験 対策`
  - practiceN: 10
  - exam: `{ enabled: true, n: 50, minutes: 30, passPct: 0.9 }`（実際の仮免学科試験に準拠）
- コードを `KARIMEN` でなく `MENKYO` にする理由: 将来第二段階の問題を追加しても localStorage 統計キー（`quizStats_MENKYO_v1` / `quizHistory_MENKYO_v1`）を維持するため。

## 問題データ（約100問・オリジナル作成）

- 実在の試験問題・教本の転載はしない。道路交通法の知識に基づくオリジナル問題を作成する。
- 形式: ○×中心（8割目安）＋4択（標識の識別、優先順位、法定数値など）。
- ○×問題は `o: ["正しい", "誤り"]` とし、新フィールド `f: true`（選択肢固定順）を付与する。
- 分野（`d`）構成（各10〜15問、仮免第一段階の教程準拠）:
  1. 信号・警察官の手信号
  2. 標識・標示
  3. 通行区分・歩行者保護
  4. 速度・追越し
  5. 交差点・優先関係
  6. 徐行・一時停止
  7. 駐停車
  8. 乗車・積載・免許制度
  9. 安全確認・運転操作
- `oe`（選択肢別解説）・`e`（総括解説）は既存フォーマット準拠。`d|t` はファイル内一意。

## エンジン改修

- `src/scripts/quiz.js`: 出題アイテム構築時（現行 219 行付近）、`q.f` が truthy なら選択肢をシャッフルせず元順のまま表示する。
- `scripts/validate.js`: 任意フィールド `f` を許容（boolean 検証）。`FILES` に `questions-menkyo.json` を追加。

## ナビ統合

- `src/components/Quiz.astro` のデックナビに `MENKYO` リンクを追加。
- `src/pages/index.astro` の decks 配列にカードを追加（note: `仮免学科 第一段階 · 100問`）。

## 検証

- `bun run validate` で構造チェック。
- `bun run dev` で ○×固定順・4択シャッフル・試験モード（50問/30分/90%）を手動確認。
- UI 変更を含む PR のため Playwright でスクリーンショットを撮影する。

## エラーハンドリング / テスト

既存エンジンの仕組みに乗るため新規のエラーパスはない。検証は validate.js と手動確認（このリポジトリの慣行どおり）。
